gg_varpro — faithful = TRUE Mode

How should per-tree importance scores be shown alongside the summary box?

A — Box + jittered per-tree points

Honest box (15/85 hinges) rendered at reduced opacity; individual tree z-scores overlaid as jittered dots; white-filled mean point. Caption notes non-Tukey boxplot.

B — Strip plot + mean ± SD bar

Box replaced by a mean ± SD interval bar (matching varPro's bxp stats exactly). Per-tree z-scores as a transparent strip. Filled dot = mean. Closer to varPro's internal rendering.

Context — faithful = TRUE

Both modes require a mandatory plot caption: "Hinges show 15th/85th percentiles; whiskers show 5th/95th. This is not a Tukey boxplot." (Option A) or "Bar shows mean ± SD from per-tree importance." (Option B). The default faithful = FALSE view (previous question) always uses the honest 15/85 box regardless of this choice.
